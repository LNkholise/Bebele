## Features

- Browse verses from both the **Old Testament** and **New Testament**
- Quickly navigate through books, chapters, and verses
- **Copy verses to your clipboard** with a single tap
- Intuitive **slider controls** for selecting books, chapters, and verses
- Fast and lightweight user experience
- Clean, mobile-friendly interface

## Installation & Running

```bash
git clone https://github.com/LNkholise/Bebele.git
cd bebele
```
```bash
npm install
```
#### To use in Expo go 

```bash
npm start
```

#### For Android use

```bash
npm android
```
For more commands, please check `package.json`.
